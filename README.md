## 🚀 What I'm Building

```
📸 Photo of Hot Wheels
        │
        ▼
🤖 Llama 4 Vision  →  Identifies exact car, rarity, condition
        │
        ▼
🧠 Kimi K2 on Groq  →  Live ₹ prices from OLX, Instagram, Maido
        │
        ▼
✅ Result in ~10 seconds — India's first tool of its kind
```

**HotScan India** is a PWA + Chrome Extension that helps 35,000+ Indian Hot Wheels collectors:
- 🔍 Identify any Hot Wheels car from a photo using AI
- 💰 Get live Indian market prices (retail + collector)
- ⭐ Detect rarity — Common → Super Treasure Hunt → Error Car
- 📦 Track their collection with cloud sync
- 📈 Investment ratings for every car
